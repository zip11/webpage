Go语言转义字符

在编程中有一些字符是打不出来的，比如换行，提示音，下一个制表位等等，于是程序语言的开发者就设计了转义序列（escape sequence）这种东西用来表达这些字符。

所谓转义，可以理解为 “采用某些方式暂时取消该字符本来的含义”，这里的 “某种方式” 指的就是在指定字符前添加反斜杠 \，以此来表示对该字符进行转义。

\t 	表示一个制表符，通常使用它可以排版
\n 	换行符
\ 	一个\
" 	一个"
\r 	一个回车

package main
import "fmt"
func main() {
	fmt.Println("Hello ")
	fmt.Println("Hello \nCoder")
}
