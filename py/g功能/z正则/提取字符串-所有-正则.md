## re.match() ##

与re.search()非常类似，主要区别就是前者是从目标字符串的开头匹配，而后者则要没有这个要求。

## re.search() ##

group方法表示正则表达式的分组：

1、在正则表达式里使用 () 表示一个分组

2、如果没有分组，默认只有一组

3、分组的下标从 0 开始

    # 正则表达式有 4 个分组
    m1 = re.search(r'(9.*)(0.*)(5.*7)', 'da9fi0rie15kfsda7ifsdaiferit')
    
    print(m1.group(0)) 
    
    # 第 0 组就是把整个表达式当做一个整体
    print(m1.group()) 
    
    # 默认就是拿第 0 组
    print(m1.group(1))  
    # 9fi

    print(m1.group(2)) 
    # 0riel
